# react-koa-sbase-boilerplate

基于 [react-koa-sbase](url=https://github.com/dongwenxiao/react-koa-sbase) 构建的模板项目


## 使用

国内用户建议使用 ```cnpm``` 或 ```yarn``` 代替 ```npm```


```
// 安装依赖
npm i

// 打包并启动
npm start

// 客户端打包
npm run client-dist

// 服务端打包
npm run server-dist
```

打开 [http://localhost:3000/about](http://localhost:3000/about) 看效果
